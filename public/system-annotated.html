13 RP 全站通則摘要
來自 RP 規格的跨模組通用規則
類別通則規則摘要
全站Token登錄後維持 72 小時。設備未變動、IP 變動 → 不失效；設備變動 → 失效。有延長機制
全站Timeout統一 20 秒（原 10 秒，2023/07/24 確認）
全站網頁重整重整後停留原有頁面（GOR-5608）。Token 失效時回到「推廣類型」設置頁面
全站金額顯示小數點第二位，第二位後無條件捨去（2026/03/03）
登錄異地登錄觸發上次 IP ≠ 本次 IP & 登記 IP ≠ 本次 IP；上次裝置 ≠ 本次裝置 & 注冊裝置 ≠ 本次裝置
全站Safari 場館外鏈Safari 限制彈出項目，需加提醒讓用戶自主選擇：「即將另開遊戲界面，是否繼續」
全站跳轉類型「開啟APP」PC 無 APP 統一用跳轉參數。H5/APP 判斷 scheme/package name 是否符合

P1 App 初始化決策樹 L2
composables/useInit.ts > initFlow() — onMounted 觸發的完整初始化序列
initFlow() 完整決策流程
onMounted
  └→ initFlow()
       ├─ 1. useDeviceUuid()                    // 取得/產生裝置指紋
       ├─ 2. fetchChannelId()
       │     ├─ query.channelId 有值 → 用它
       │     ├─ settingStore.channelId 已有 → return
       │     ├─ isLocalhost → 用 env VITE_CHANNEL_ID
       │     └─ else → POST /config/getH5Domain { domain } → 取 channelId
       │     └─ 都沒有 → throw 'fetchChannelId Fail!'
       │
       ├─ 3. fetchConfig()                       // 並行拉兩支 API
       │     ├─ Promise.allSettled([
       │     │    baseConfig/v2 { channelId, deviceId, devicePlatform },
       │     │    channelIpBlackList/v3 { channelId, deviceId }
       │     │  ])
       │     ├─ 兩者都成功 → 寫入 settingStore
       │     └─ 任一失敗 → throw 'fetchConfig Fail'
       │
       ├─ 4. DeBox 偵測（navigator.userAgent 含 'DeBox'）
       │     └─ 是 DeBox → deboxLoginRegister → setToken + getUserInfo
       │
       ├─ 5. initCnnzCode()                      // 初始化統計碼
       ├─ 6. Token 續期（非 DeBox 且 hasToken）
       │     └─ refreshToken → getUserInfo()
       │     └─ 失敗 → console.warn（不中斷）
       ├─ 7. fetchIngService()                   // 進行中客服
       ├─ 8. 非維護中 → openAppHitPoint + fetchGameList
       └─ 9. setIsInitSuccess(true)

  [catch] → toast「初始化失败」+ 300ms 後 localStorage.clear()
  [finally] → checkSystemStatusAndRoute() + isInit=true + loading=false

帳號登入流程（AccountLogin）
使用者輸入帳號/密碼
  → vee-validate + zod 前端驗證
  → loginByAccount()
     ├─ 密碼 AES 加密（passwordEncrypt）→ isEncrypt: true
     ├─ POST /login/login { ...baseData, account, password, fromDomain, keyCode?, validCode? }
     ├─ errCode === 1
     │    ├─ isRememberAccount → localStorage 記住帳號+加密密碼
     │    ├─ handleLoginSuccess → setToken + 關閉彈窗 + setLoggedInBefore
     │    └─ loading 由後續 getUserInfo 關閉
     └─ errCode !== 1 → handleError（見錯誤碼表）

  → loginByPhone()

     ├─ 成功 → handleLoginSuccess
     └─ 失敗 → handleError
