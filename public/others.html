12 跨服務限制

技術限制

客服 iframe 跨域 postMessage 通訊
描述
官方客服透過 postMessage 通訊（7 種 action），跨域環境下偶爾 message 遺失。
影響
測試域名未在白名單內會導致客服功能異常。
PM 對策
新環境上線前確認客服 iframe domain 已加入白名單。
爆料詳情雙重請求設計
描述
detail（盤口+metadata）和 content（HTML 富文本）分開請求。
影響
首次載入稍慢，但降低主請求回應大小。
PM 對策
有意設計，非 bug。
客服 iframe 全局持久不卸載
描述
掛在 app layout 層，不隨頁面切換卸載。全螢幕橫屏時隱藏。
影響
常駐佔用記憶體；橫屏遊戲中無法使用客服。
PM 對策
已知行為。

模組耦合

爆料入口散布多個模組
描述
爆料入口在首頁、直播間、賽事詳情等，各入口的訪客檢查未統一。
影響
部分入口可能遺漏 gamePredictVisitorDisplayMode 檢查。
PM 對策
新增爆料入口需確認訪客攔截。
跟單投注整合體育購物車
描述
爆料跟單透過 formatCartData 格式化後加入體育購物車。
影響
購物車結構改動會影響爆料跟單。
PM 對策
體育購物車重構需同步測試爆料跟單。
消息中心未讀數由 popupStore 驅動
描述
公告未讀數從 popupStore.getNotReadNoticeCount 同步到 Tab badge。
影響
popupStore 初始化延遲會導致 badge 數字暫時為 0。
PM 對策
已知行為。

設定層限制

客服 VIP 線道分流由後台設定
描述
getCsLineConfig 回傳的線道含 vipBegin~vipEnd，前端依此過濾。
影響
VIP 升降級後線道切換不即時（快取值）。
PM 對策
VIP 相關客服問題需後台調整。
gamePredictEnabled 控制爆料開關
描述
後台關閉時，爆料首頁 onMounted 直接跳轉 /live。
影響
整個爆料模組一鍵關閉。
PM 對策
關閉前確認不影響活躍用戶。
gamePredictVisitorDisplayMode 訪客模式
描述
mode=1 允許訪客瀏覽，其他值攔截。
影響
部分入口未統一檢查此開關。
PM 對策
確認所有爆料入口的訪客邏輯。

業務限制

落地頁新舊用戶判斷依賴打點記錄
描述
新/舊用戶判斷是否曾下載過 APP，依打點記錄。
影響
清除瀏覽器資料後可能誤判為新用戶。
PM 對策
打點記錄不可跨瀏覽器/裝置。
URL 簽名用 MD5 加密
描述
客服 URL 的 sign 用 MD5(uid+agentid+mac+uname+avatar+theme+kefuSign)。
影響
kefuSign 是 runtime config 環境變數，不同環境值不同。
PM 對策
新環境需確認 kefuSign 已設定。
爆料草稿管理已暫停
描述
GOR-5554 草稿管理曾規劃但未上線。
影響
主播無法儲存草稿，編輯中離開即遺失。
PM 對策
如需重啟需重新開需求。
