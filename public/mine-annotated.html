1 核心流程
我的頁面的主要操作流程與進入路徑
我的頁面載入流程
底部 Tab「我的」 → 判斷登入狀態 → 未登入：顯示「您还未登录」
↓
已登入 → 載入用戶資訊卡 → walletStore.getUserGold() → 顯示餘額 + VIP + 稱號
快捷 Menu 路由
立即存款 → deposit 我要提款 → withdraw
立即轉帳 → mine/transfer 邀請好友 → mine/invite-friends
Badge / 紅點顯示邏輯
消息中心紅點 → notReadNoticeCount + messageNum + activityNum > 0
福利中心 badge → unclaimedWelfareCount
進行中訂單 → OrderInfo 組件（存款/提款進度）
2 6.1 我的主頁面
玩家個人中心首頁 · pages/mine/index.vue
已登入狀態（含進行中訂單）
未登入狀態
logic

登入狀態判斷

未登入 → 顯示「您还未登录」+ 點擊頭像跳登入頁

已登入 → 顯示 loginAccount + 總餘額

ui

用戶資訊卡

頭像：依 iconCustom + 主題（dark/light）從 CDN 載入

VIP 圖示：vipStore.vipIcons[vipLevel].basicIcon

稱號 badge：badgeInfo.img 從 CDN 載入稱號圖片

api

餘額刷新機制

walletStore.getUserGold() + 10 秒冷卻

餘額計算：walletStore.totalBalance / 100

pages/mine/index.vue → startCountdown(10)
warning

測試帳號限制

isTestAccount 時隱藏存款/提款/轉帳等金流功能（disallowTest: true）

ui

每日任務區

MineDailySign（簽到）+ MineMission（任務提示）+ DailyRewardPopup

ui

福利專員入口

WelfareCommissioner 組件在左上角，特定條件下顯示浮動入口

logic

開房對賭入口

settingStore.showCommunityRoomHome 開啟時，頁面最底部顯示 OpenRoomEntrance

首次進入自動滾動到該位置

3 6.1.1 個人資料
個人中心 · pages/mine/mine-center/index.vue
個人資料頁
稱號主頁
logic

別名（Alias）三種狀態

NO_SET(0) → 未設定

CONFIRMING(1) → 審核中

SUCCESS_SET(3) → 已設定

修改限制：aliasConfig.changeAliasEnable + changeAliasInterval（間隔天數）

validation

生日只能設定一次

BirthdayPicker：預設當日，年份區間 100 年（e.g. 1925-01-01 ~ 2025-01-01）

flow

子路由一覽

head-photo / bind-phone / bind-email / bind-qq / bind-wechat / change-password / real-name / badgeInfo / aliasInfo-name

pages/mine/mine-center/*
flow

登出

useLogout() + userStore.logout() → 跳登入頁

ui

RP 通則（VIC-102）

確認按鈕文案統一從「確認」改為「確定」

非金額數字統一使用非 Akrobat 字體

16 6.15 福利中心
福利領取中心 · pages/mine/welfare-center/
福利中心主頁
待領取確認彈窗
logic

資料邏輯

useWelfareCalendar() — 日期範圍選擇

領取動作

logic

獎品分類

curFilterCategoryIsAward：判斷當前篩選是否為獎品分類

獎品分類時顯示獎品分類圖示（getAwardCategoryImageUrl）
